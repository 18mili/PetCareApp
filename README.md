# PetCare App

Proyecto desarrollado en **Kotlin + Jetpack Compose (Material3)**  
para la Evaluación 2 de la asignatura **DSY1105 - Aplicaciones Móviles**.

##  Requerimientos técnicos
- Jetpack Compose + Material 3  
- Navegación funcional entre pantallas  
- Formularios con validación  
- Almacenamiento local con DataStore (opcional extra)  
- Proyecto Android completo y ejecutable  
- Repositorio GitHub y APK adjunto  

##  Ejecución
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/18mili/PetCareApp.git
2.Abrir el proyecto en Android Studio (versión Ladybug o superior).
3.Ejecutar con ▶️ en emulador o dispositivo físico.

   Funcionalidades
  -Inicio de sesión y registro de usuarios.
  -Validación de formularios.
  -Registro de mascotas (nombre, especie, edad).
  -Persistencia local con DataStore.
  -Cierre de sesión y navegación entre pantallas.

  APK

Archivo disponible en la raíz del proyecto:
app-debug.apk
